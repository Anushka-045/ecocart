*ECOCART AI* — an AI-powered sustainability advisor built to bring clarity, transparency, and real environmental insights to online shopping.

EcoCart AI intelligently analyzes product sustainability in real time by evaluating materials, packaging impact, eco-certifications, and detecting greenwashing claims. It generates a clear Eco Score, sustainability verdict, positive and negative environmental signals, and actionable recommendations, enabling consumers to make informed and eco-conscious purchasing decisions.

🔥 *Major Backend Upgrade – Auto-Scraping Feature*

*Our Flask backend has been enhanced with an automatic product scraping system.*

Earlier, product analysis required manual input of product title and description. *Now, users can simply enter a product URL, and EcoCart AI automatically extracts the necessary details for sustainability analysis.*

Manual input is still supported as a fallback for sites that restrict automated scraping, ensuring reliability and accessibility for all users.

*⚙ How It Works*
• User enters a product URL or product description
• Backend auto-extracts product details (title, description)
• AI evaluates environmental impact via OpenRouter
• System formats insights into clean JSON
• Frontend displays real-time sustainability results

🌍 *Why EcoCart AI Matters*
Today’s e-commerce is filled with vague eco claims and hidden greenwashing. Users lack reliable tools to evaluate the true environmental impact of products. EcoCart AI fills this gap with real-time AI-driven sustainability insights, empowering users to shop smarter and greener.
